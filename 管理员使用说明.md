# 管理员使用说明

## 🔐 管理员控制台

### 访问地址
```
http://localhost:1236/admin
```

### 默认密码
```
admin123
```

**重要**：首次使用后，建议修改密码（在 `server_multiplayer.py` 第29行修改 `ADMIN_PASSWORD`）

## 📋 使用流程

### 1. 启动服务器
双击运行 `启动多人服务器.bat`

### 2. 打开管理员控制台
在浏览器中访问：`http://localhost:1236/admin`

### 3. 登录管理员
- 输入密码：`admin123`
- 点击"登录"按钮

### 4. 等待用户加入
- 查看"在线用户"数量
- 等待所有学生/参与者加入

### 5. 开始游戏
- 点击"开始游戏"按钮
- 系统会自动进行5秒倒计时
- 倒计时结束后，游戏正式开始

### 6. 游戏进行中
- 实时查看游戏状态
- 查看市场价格变化
- 查看在线用户数量

### 7. 重置游戏（可选）
- 游戏结束后，可以点击"重置游戏"按钮
- 这将清除所有用户的交易数据
- 可以重新开始新一轮游戏

## 🎯 功能说明

### 管理员控制台功能

1. **登录验证**
   - 需要输入正确的管理员密码
   - 密码错误会提示

2. **实时状态监控**
   - 游戏状态（等待开始/倒计时中/游戏中）
   - 在线用户数量
   - 游戏剩余时间
   - 当前市场价格

3. **游戏控制**
   - **开始游戏**：启动5秒倒计时，然后开始游戏
   - **重置游戏**：清除所有数据，回到初始状态

4. **状态显示**
   - 倒计时数字实时显示
   - 游戏状态颜色标识：
     - 🟡 等待开始（黄色）
     - 🔴 倒计时中（红色）
     - 🟢 游戏中（绿色）

## ⚠️ 注意事项

1. **密码安全**
   - 默认密码是 `admin123`
   - 建议修改为更安全的密码
   - 修改方法：编辑 `server_multiplayer.py` 第29行

2. **游戏开始**
   - 只有管理员可以启动游戏
   - 用户端会显示"等待管理员启动"
   - 管理员点击"开始游戏"后，所有用户会看到倒计时

3. **重置游戏**
   - 重置会清除所有用户的交易数据
   - 需要确认操作
   - 重置后可以重新开始

4. **多管理员**
   - 支持多个管理员同时登录
   - 任何管理员都可以控制游戏

## 🔧 修改管理员密码

编辑 `server_multiplayer.py` 文件，找到第29行：

```python
ADMIN_PASSWORD = 'admin123'  # 管理员密码，可以修改
```

修改为您想要的密码，例如：

```python
ADMIN_PASSWORD = 'your_secure_password_here'
```

保存后重启服务器即可生效。

## 📱 学生/用户端

学生访问：`http://localhost:1236`（或服务器IP地址）

- 输入昵称
- 看到"等待管理员启动"提示
- 等待管理员开始游戏
- 看到5秒倒计时
- 开始交易

## 🎓 教学场景建议

1. **课前准备**
   - 提前启动服务器
   - 打开管理员控制台
   - 等待所有学生加入

2. **开始实验**
   - 确认所有学生已加入
   - 点击"开始游戏"
   - 观察学生交易行为

3. **实验结束**
   - 游戏自动结束（280秒）
   - 可以查看最终结果
   - 如需重新开始，点击"重置游戏"

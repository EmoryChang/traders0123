# 多人实时模拟交易平台 - 完整启动指南

## 📋 第一步：检查Python环境

### 1.1 检查是否已安装Python

打开命令提示符（CMD）或PowerShell，输入：
```bash
python --version
```

如果显示Python版本（如 `Python 3.8.0`），说明已安装，可以继续下一步。

如果没有安装Python：
- 访问 https://www.python.org/downloads/
- 下载并安装Python 3.7或更高版本
- **重要**：安装时勾选 "Add Python to PATH"

## 📦 第二步：安装依赖包

### 方法1：使用批处理文件（推荐，最简单）

1. 双击运行 `安装依赖.bat` 文件
2. 等待安装完成（可能需要几分钟）
3. 看到 "安装完成！" 提示后，按任意键关闭

### 方法2：使用命令行

1. 打开命令提示符（CMD）或PowerShell
2. 进入项目目录：
   ```bash
   cd "D:\AI\Traders\无联动"
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

**如果遇到权限错误**，尝试：
```bash
pip install --user -r requirements.txt
```

## 🚀 第三步：启动服务器

### 方法1：使用批处理文件（推荐）

1. 双击运行 `启动多人服务器.bat` 文件
2. 会看到类似以下信息：
   ```
   ========================================
   多人实时模拟交易平台服务器
   端口: 1236
   支持最多100+用户同时在线
   ========================================
   ```
3. **保持这个窗口打开**，不要关闭它

### 方法2：使用命令行

1. 打开命令提示符（CMD）或PowerShell
2. 进入项目目录：
   ```bash
   cd "D:\AI\Traders\无联动"
   ```
3. 启动服务器：
   ```bash
   python server_multiplayer.py
   ```

## 🌐 第四步：访问平台

### 本地访问

1. 打开浏览器（Chrome、Edge、Firefox等）
2. 在地址栏输入：
   ```
   http://localhost:1236
   ```
3. 按回车键访问

### 局域网访问（让其他设备访问）

1. 查看服务器电脑的IP地址：
   - 打开命令提示符
   - 输入：`ipconfig`
   - 找到 "IPv4 地址"，例如：`192.168.1.100`

2. 在其他设备（手机、平板、其他电脑）的浏览器中输入：
   ```
   http://192.168.1.100:1236
   ```
   （将 `192.168.1.100` 替换为您的实际IP地址）

## 🎮 第五步：开始使用

1. **输入昵称**：首次访问时，会弹出昵称输入框
   - 输入您的昵称（例如：学生001）
   - 点击"确定"或按回车键

2. **等待倒计时**：所有用户会看到5秒倒计时（5、4、3、2、1）

3. **开始交易**：倒计时结束后，点击买入/卖出按钮进行交易

## ⚠️ 常见问题

### 问题1：提示"pip不是内部或外部命令"
**解决方法**：
- 确保Python已正确安装并添加到PATH
- 尝试使用 `python -m pip install -r requirements.txt`

### 问题2：端口1236已被占用
**解决方法**：
- 关闭占用1236端口的其他程序
- 或者修改 `server_multiplayer.py` 中的端口号（第442行）

### 问题3：无法访问服务器
**解决方法**：
- 检查防火墙是否允许1236端口
- 确保服务器窗口没有关闭
- 检查浏览器地址是否正确

### 问题4：依赖安装失败
**解决方法**：
- 确保网络连接正常
- 尝试使用国内镜像源：
  ```bash
  pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```

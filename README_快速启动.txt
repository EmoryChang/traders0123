========================================
多人实时模拟交易平台 - 快速启动指南
========================================

【第一步】检查Python
-------------------
打开命令提示符，输入: python --version
如果显示版本号，说明已安装，继续下一步
如果没有，请先安装Python 3.7+ (https://www.python.org/downloads/)

【第二步】安装依赖
-------------------
双击运行: 安装依赖.bat
等待安装完成（可能需要几分钟）

【第三步】启动服务器
-------------------
双击运行: 启动多人服务器.bat
保持窗口打开，不要关闭

【第四步】访问平台
-------------------
在浏览器中打开: http://localhost:1236

【第五步】开始使用
-------------------
1. 输入您的昵称
2. 等待5秒倒计时
3. 开始交易！

========================================
局域网访问（让其他设备访问）
========================================

1. 查看服务器IP地址:
   打开命令提示符，输入: ipconfig
   找到 "IPv4 地址"，例如: 192.168.1.100

2. 在其他设备浏览器输入:
   http://192.168.1.100:1236
   (替换为您的实际IP地址)

========================================
常见问题
========================================

Q: 提示"pip不是内部或外部命令"
A: 确保Python已安装并添加到PATH，或使用: python -m pip install -r requirements.txt

Q: 端口1236已被占用
A: 关闭占用该端口的程序，或修改server_multiplayer.py中的端口号

Q: 无法访问服务器
A: 检查防火墙设置，确保1236端口未被阻止
